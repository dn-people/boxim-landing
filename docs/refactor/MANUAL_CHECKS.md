# 수동 육안 체크 (구조 변경 항목마다 npm start 후 수행)

1. 히어로: 배경 확대→축소 애니메이션, 타이틀 3종("스마트폰 구매/통신사 가입/유심 구매") 2초 간격 순환
2. 헤더: 어두운 섹션(히어로/회사/연혁/마지막)에서 검정 배경+흰 글자, 밝은 섹션에서 흰 배경+회색 글자로 전환
3. 헤더 탭: 스크롤 위치에 따라 홈/소개/회사/후기 활성(파란 배경) 이동, "블로그" 클릭 시 /blog/ 이동
4. 소개: 이용자 사례 3종 5초 간격 순환
5. 미션: 스크롤 진행 시 첫 카드가 위로 사라지며 두 번째 카드 노출(sticky)
6. 상품: 8개 아이콘 카드가 순차 등장(그리드 2열/4열)
7. 회사: 통계 3카드, 연혁 3카드 반투명 블러 배경
8. 후기: 마키 2줄이 서로 반대 방향으로 무한 스크롤, 별 5개씩
9. 마지막: 배경 축소 애니메이션 + "바로가기" 버튼 → boxim.io 새 탭
10. 콘솔: 에러 0건 (경고 중 React DevTools 안내는 무시)
