<h2>Q1. 알뜰폰 번호이동은 무엇을 바꾸는 건가요?</h2>
<p>알뜰폰 번호이동은 전화번호는 그대로 두고 이용 통신사와 요금제만 바꾸는 절차입니다. 새 번호를 받는 신규가입과 다르고, 같은 통신사 안에서 단말기만 바꾸는 기기변경과도 다릅니다.</p>
<p>예를 들어 기존 번호를 계속 쓰면서 SKT, KT, LG U+ 망을 쓰는 알뜰폰 요금제로 옮기는 경우가 번호이동입니다. 이때 중요한 기준은 이름이 비슷한 요금제가 아니라 통신망, 데이터 제공량, 할인 종료 후 요금, 유심 또는 eSIM 개통 가능 여부입니다.</p>

<h2>Q2. 신청 전에 무엇을 확인해야 하나요?</h2>
<p>신청 전에는 기존 회선의 약정, 할부금, 결합 할인, 본인인증 가능 여부를 먼저 확인해야 합니다. 번호이동은 새 통신사에서 개통이 완료되면 기존 회선이 정리되는 흐름이므로, 남아 있는 비용과 혜택이 있는지 먼저 보는 것이 안전합니다.</p>

<h2>Q3. 알뜰폰 번호이동 절차는 어떤 순서인가요?</h2>
<p>알뜰폰 번호이동은 요금제 선택, 가입 신청, 번호이동 인증, 개통 요청, 유심 장착 순서로 진행됩니다. 사업자마다 화면 이름은 다를 수 있지만, 큰 흐름은 비슷합니다.</p>
<ol>
<li>현재 쓰는 번호, 명의자, 기존 통신사 정보를 확인합니다.</li>
<li>데이터 사용량과 통신망을 기준으로 알뜰폰 요금제를 고릅니다.</li>
<li>신분증, 본인 명의 결제수단, 본인인증 수단을 준비합니다.</li>
<li>유심 배송형인지, 편의점 유심인지, eSIM 개통인지 확인합니다.</li>
<li>가입 신청 화면에서 번호이동을 선택하고 기존 번호를 입력합니다.</li>
<li>번호이동 인증 문자를 확인하거나 안내된 인증 절차를 완료합니다.</li>
<li>개통 완료 안내를 받은 뒤 새 유심을 장착하거나 eSIM을 등록합니다.</li>
<li>통화, 문자, 데이터가 정상 작동하는지 확인하고 필요하면 재부팅합니다.</li>
</ol>
<p>알뜰폰 요금제 자체를 비교할 때는 <a href="https://www.mvnohub.kr/" target="_blank" rel="noopener">알뜰폰허브</a> 같은 공식 비교 서비스를 참고할 수 있습니다. 단, 신청 최종 조건은 실제 가입 화면에서 다시 확인해야 합니다.</p>

<h2>Q4. 셀프개통과 상담원 개통은 무엇이 다른가요?</h2>
<p>셀프개통은 사용자가 온라인에서 직접 개통 요청을 하는 방식이고, 상담원 개통은 사업자 확인을 거쳐 처리되는 방식입니다. 빠르게 끝나는 경우도 있지만, 본인인증이나 번호이동 인증이 막히면 상담이 필요한 경우가 있습니다.</p>
<p>처음 번호이동을 한다면 셀프개통 가능 시간, 취소 가능 여부, 유심 배송 상태를 미리 확인하세요. 개통 중에는 기존 회선과 새 회선이 전환되는 과정이 있으므로 업무용 번호라면 통화가 덜 필요한 시간에 진행하는 편이 좋습니다.</p>

<h2>Q5. 유심과 eSIM은 어떻게 준비해야 하나요?</h2>
<p>유심은 실제 칩을 핸드폰에 넣는 방식이고, eSIM은 지원 단말에서 QR이나 앱으로 회선을 등록하는 방식입니다. 쓰는 핸드폰이 eSIM을 지원하지 않으면 유심 개통을 선택해야 합니다.</p>
<p>배송 유심은 받기 전까지 개통이 진행되지 않는 경우가 있고, 편의점 유심은 구매한 유심 번호를 신청 화면에 입력해야 하는 경우가 있습니다. eSIM은 빠르게 열 수 있지만, 단말기 초기화나 기기 변경 때 재발급 조건이 있을 수 있으므로 안내 문구를 확인해야 합니다.</p>

<h2>Q6. 기존 통신사는 직접 해지해야 하나요?</h2>
<p>번호이동은 새 통신사 개통이 완료되면 기존 회선이 정리되는 흐름이 일반적입니다. 그래서 먼저 기존 통신사를 해지해 버리면 쓰던 번호를 옮기지 못할 수 있습니다.</p>
<p>직접 해지가 필요한지 헷갈린다면 가입하려는 알뜰폰 사업자 안내를 기준으로 보세요. 특히 가족 결합, 인터넷 결합, 멤버십, 소액결제, 부가서비스가 연결돼 있다면 번호이동 뒤 어떤 항목이 종료되는지 기존 통신사에도 확인하는 것이 좋습니다.</p>

<h2>Q7. 번호이동 후 통화나 데이터가 안 되면 어떻게 하나요?</h2>
<p>개통 뒤 통화나 데이터가 안 되면 유심 장착 상태, 재부팅, 네트워크 설정을 차례로 확인하세요. 유심을 여러 번 뺐다 끼우기보다 기본 점검을 한 번씩 진행하는 편이 낫습니다.</p>
<p>데이터만 안 된다면 APN 설정 안내가 필요한 상품인지 확인하세요. 문자 인증이 안 오거나 통화가 전혀 되지 않으면 개통 완료 상태가 아닌 경우도 있으므로, 신청한 사업자 고객센터에서 회선 상태를 확인해야 합니다.</p>

<p>알뜰폰 번호이동은 절차 자체보다 신청 전 확인이 더 중요합니다. 기존 통신사의 남은 비용, 새 요금제의 할인 종료 후 금액, 유심 또는 eSIM 개통 방식, 단말기 상태를 함께 보면 개통 뒤 당황할 일이 줄어듭니다.</p>
